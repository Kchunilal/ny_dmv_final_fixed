# New York DMV Driver Points Analysis

**Project Overview**  
Python scripts to analyze and process New York State DMV driver violation points, with a focus on recent changes to the point system. This project calculates current point totals, tracks point accumulation/reduction over time, identifies suspension risks, and visualizes the impact of the latest rule updates.

### Key Features
- Calculates current and historical driver points with automatic 18-month expiration logic
- Applies the most recent NY DMV point changes and rule updates
- Identifies drivers at risk of license suspension (11+ points rule, provisional licenses, etc.)
- Analyzes violation types, frequency, and severity
- Generates clear summary reports and visualizations
